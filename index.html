Form

Bootstrap's animated & overlayed form elements

The base about form elements the structure still the bootstrap structure, with a .form-group composed of a label and a input.form-control. 

<div class="form-group">
    <label>Example 1</label>
    <input class="form-control" type="text">
</div>

For using the animated version, just add the variant class .-animated on the .form-group element 

<div class="form-group -animated">
    <label>Example 2</label>
    <input class="form-control" type="text">
</div>

To put an global overlay on the focus of the .form-control, use the variant class .-overlayed. 

<div class="form-group -overlayed">
    <label>Example 3</label>
    <input class="form-control" type="text">
</div>

You can of course use .-animated and .-overlayed on the same element 

<div class="form-group -animated -overlayed">
    <label>Example 4</label>
    <input class="form-control" type="text">
</div>

If you need to you can add a form-caption that will appear when the input is focused. You need to add a third element into the .form-group with the class code{.form-caption} and then whatever you want into.

Lorem ipsum
<div class="form-group -animated -overlayed">
    <div class="form-caption">lorem ipsum</div>
    <label>Example 5</label>
    <input class="form-control" type="text">
</div>

You can choose the position of the form-caption (the 4 corner) with the variant classes .-topLeft, .-topRight, .-bottomRight, .-bottomLeft. For Mobile and Tablet (xs and sm viewport), you have the choice between 2 position top or bottom with the respective classes -touchTop and -touchBottom
